# 📚 BookVill — Read Everything. All in One Place.

> Novels · Manga · Manhwa · Manhua · Comics · Light Novels

---

## 🔍 What Problem Are We Solving?

Readers today are **scattered across dozens of platforms** just to enjoy different formats:
- **Novels** → Kindle, Wattpad, Royal Road
- **Manga** → MangaDex, MangaPlus
- **Manhwa** → Webtoon, TappyToon
- **Manhua** → Bilibili Comics, MangaToon

This fragmentation means:
- Managing **multiple accounts** and subscriptions
- **Inconsistent reading experiences** across apps
- No **unified library** or reading history
- Missing out on **cross-format discovery** ("If you liked this manga, try this novel")

**BookVill solves this by being the single destination for all reading formats — one account, one library, one reader.**

## 🚀 Core Features (v1.0)

### 1. 👤 User Authentication
- Sign up / Login (Email + Password)
- OAuth (Google, Discord)
- User profile with reading stats

### 2. 📚 Book Library
- Browse all available titles
- Organized by format (Novel, Manga, Manhwa, Manhua)
- Cover thumbnails, title, author, rating, chapter count
- "Continue Reading" section on homepage

### 3. 📖 Reader
- **Novel mode** → Paginated or scrolling text reader
- **Manga mode** → Panel-by-panel right-to-left reading
- **Manhwa mode** → Infinite vertical scroll
- Font size, brightness, and theme controls
- Chapter navigation (Prev / Next)

### 4. 🔖 Bookmark & Progress Tracking
- Bookmark specific pages / chapters
- Auto-save reading progress
- Reading history timeline
- Mark as: Reading / Completed / Plan to Read / Dropped

### 5. 🗂️ Categories & Discovery
- Genre tags: Fantasy, Romance, Action, Mystery, Horror, etc.
- Format filter: Novel / Manga / Manhwa / Manhua / Comics
- Trending, New Arrivals, Top Rated sections
- Recommended For You (based on reading history)

### 6. 🔎 Search
- Full-text search by title, author, genre
- Filter by format, status (Ongoing / Completed), rating
- Autocomplete suggestions
